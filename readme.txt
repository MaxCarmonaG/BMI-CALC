Título: BMI-CALC
Descripción: Aplicación para android que permite el cálculo del índice de masa corporal (IMC) y entrega como respuesta recomendaciones para el bienestar del usuario. Adicional permite la generación de registros en una base de datos MySQL para posterior consulta. La aplicación contará con 2 roles, usuario y administrativo los cuales condicionalmente tendrán acceso a diferentes vistas y funcionalidades.
Adicional se implementarán formularios para la creación de usuarios y de autenticación para usuarios registrados.
El rol de usuario permitirá crear registros y consultar el histórico de registros. El rol administrativo permitirá ver los usuarios registrados e informes estadísticos de dicha población.
Clave para las key signature es 123456.
